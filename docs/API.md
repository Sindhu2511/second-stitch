# API Reference

Base URL: `http://YOUR_SERVER:8080`

---

## Endpoints

### `GET /api/health`

Returns GPU and model status.

**Response:**
```json
{
  "status": "ok",
  "gpu": "NVIDIA RTX A6000",
  "classifier": "ready",
  "sd_pipeline": "ready"
}
```

---

### `POST /api/upcycle`

Main endpoint. Accepts a garment image, classifies it, transforms it, and returns the upcycled result.

**Request:**
- Content-Type: `multipart/form-data`
- Field: `file` — image file (JPEG, PNG, or WebP, max 10MB, min 50×50px)

**Example (curl):**
```bash
curl -X POST http://localhost:8080/api/upcycle \
  -F "file=@garment.jpg" \
  | python3 -m json.tool
```

**Example (JavaScript/React):**
```javascript
const formData = new FormData();
formData.append("file", blob, "garment.jpg");

const response = await fetch(`${API_URL}/api/upcycle`, {
  method: "POST",
  body: formData,
});
const data = await response.json();
```

**Response (200 OK):**
```json
{
  "success": true,
  "detected_garment": "short sleeve dress",
  "confidence": 0.7007,
  "upcycle_name": "Mini Dress",
  "description": "Hemline shortened to mini",
  "original_url": "/uploads/orig_abc123.jpg",
  "generated_url": "/uploads/gen_def456.jpg"
}
```

**Response fields:**

| Field | Type | Description |
|-------|------|-------------|
| `success` | bool | Whether generation succeeded |
| `detected_garment` | string | Classified garment category |
| `confidence` | float | Classifier confidence (0–1) |
| `upcycle_name` | string | Name of the transformation |
| `description` | string | Human-readable description |
| `original_url` | string | Path to saved original image |
| `generated_url` | string | Path to upcycled image |

**Error responses:**

| Code | Reason |
|------|--------|
| 400 | Invalid file type / file too large / image too small |
| 500 | Server error during processing |

---

### `GET /uploads/{filename}`

Serves generated images as static files.

**Example:**
```
GET /uploads/gen_def456.jpg
```

---

### `GET /docs`

Interactive Swagger UI documentation (auto-generated by FastAPI).

```
http://YOUR_SERVER:8080/docs
```

## Processing Time

| Step | Approx Time |
|------|-------------|
| Classification (EfficientNet-B3) | ~0.1s |
| Segmentation (SegFormer) | ~0.5s |
| Geometric transformation | ~0.05s |
| SDXL hem finishing | ~15–25s |
| **Total** | **~20–30s** |

First request after server start takes longer as models are loaded into GPU memory (~30–60s warmup).
